# Grocery-Price-Analysis-with-Numpy-Pandas-and-Matplotlib

Simple Python project using **Pandas** and **NumPy** to analyze grocery data.

## Features
- Load CSV dataset
- Calculate total rows & columns
- Compute average price per product
- Group by product type for insights

https://www.kaggle.com/datasets/acvingonsalves/grocery-dataset-for-perth/data
